Investor & Founder Thinking

April 2025

If you're an investor, think like a founder.

If you're a founder, think like an investor.

Hear me out:

If an investor thinks like a founder, they'd be able to understand the intricacies of building a project beyond the financials.

If a founder thinks like an investor, they'd become more skeptical and be able to identify any pain points—playing devil's advocate.

Think outside of your typical parameters and you'll be able to develop novel ideas/questions.
